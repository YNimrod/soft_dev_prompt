# Phase 4: コンテキストマップ作成

## プロンプト

```
## コンテキストマップ作成

{{ドキュメントルート}}/inception/units/ に作成された Unit of Work の md 設計ファイルを参照し、
各ユニットの関係性を表すコンテキストマップを作成してください。

さらに、ユニット間の動的なやり取りが時系列に分かるよう、
代表的なユーザーストーリーを用いてシーケンス図を示してください。

図は Mermaid 記法で、コントラストをはっきりさせて記述してください。
```

---

## 使用する変数

| 変数 | 説明 | 例 |
|---|---|---|
| `{{ドキュメントルート}}` | ドキュメントの保存先ルートパス | `project-docs` |
