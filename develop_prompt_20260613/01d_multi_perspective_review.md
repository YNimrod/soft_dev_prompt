# Phase 1d: 多角レビュー（専門家ペルソナ）※オプション

## このフェーズの目的

Phase 1c（アクターレビュー）でアクター視点の確認を終えた後、**ビジネス・技術・法務など外部専門家の視点**でユーザーストーリーを評価します。
評価対象によって不要なペルソナは外せます。外部レビュー自体が不要な場合はフェーズ全体をスキップできます。

---

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 将来の拡張（Phase2以降）と矛盾する設計制約がないか

ペルソナ 4: ⚖️ 法務・コンプライアンスアドバイザー（略称: LEGAL）

- UGC（ユーザー生成コンテンツ）を扱う場合、通報・削除フローがストーリーに含まれているか

## Questionフォーマットの説明

このフェーズから使用する強化版 `[Question]` フォーマットは、**なぜ今決める必要があるか・選択肢は何か・影響範囲はどこか** を明示することで、回答者（あなた）が判断しやすくなるよう設計されています。

```
[Question ペルソナ略称-番号]
・問い   : 何を決める必要があるか（一文で）
・背景   : なぜこれが未決定なのか、なぜ問題になりうるか
・選択肢 : A案（メリット/デメリット）/ B案（メリット/デメリット）
・影響   : 未決定のままだと〇〇フェーズの〇〇が進められない
[Answer ペルソナ略称-番号]
（回答をここに記入）
```
