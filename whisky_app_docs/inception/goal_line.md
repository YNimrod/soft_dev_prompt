# ゴールライン定義（Phase 0b 確定版）

## PRFAQ

ウイスキーを趣味として日々飲んでいる人は、自分が「持っている」「飲んだことがある」「飲みたい」
ボトルをスマホのメモやSNSの過去投稿で散らばったまま管理していることが多い。
新アプリ「（仮）Whisky Life Log」は、この3つの軸でボトルを記録し、テイスティングノートを残し、
見栄えのいいカードとしてSNSに発信できる。ユーザーは飲んだ瞬間の感動を逃さず記録に残し、
自分のウイスキー人生の足跡を可視化しながら、ウイスキー好きの仲間に向けて手軽に発信できるようになる。

## コアジャーニー

1. ボトルを登録する（持っている／飲んだことがある／飲みたい、のいずれかのリストに追加）
2. 飲んだ記録・テイスティングノートを残す
3. その記録を見栄えのいいカードとしてSNSに共有する

## 各ステップの最小実装（ゴールライン）

| ステップ | 最小実装 |
|---|---|
| 1. ボトル登録 | カメラで1本のボトルを撮影し画像認識で登録できる（手動入力も可）。3リストのいずれかに追加できる |
| 2. 記録 | 1回のテイスティングについて、評価項目への入力とひとことコメントを保存・一覧表示できる |
| 3. SNS共有 | 1件のテイスティング記録を画像カード化し、外部SNSへ共有できる |

## ゴールラインに含めないもの（後回し）

優先度の高低にかかわらず、コアジャーニーの成立に必須ではないため後回しとする。
「後回し」は優先度の低さを意味しない（缶ハイボール・未開栓区別は高優先度の後回し）。

| 項目 | 優先度 | 備考 |
|---|---|---|
| 一枚の写真から複数本のボトルを検出する機能 | 高 | ゴールライン達成後すぐに追加 |
| 缶ハイボールへの対応 | 高 | ゴールライン達成後すぐに追加 |
| 「持っている」リスト内の未開栓／抜栓済みの区別 | 高 | ゴールライン達成後すぐに追加 |
| Profile（統計可視化） | 中 | MVP範囲内だがゴールライン外 |
| Map（蒸留所・バー・イベント記録） | 低 | Phase 2以降 |
| ヘルス機能 | 低 | 将来拡張 |
| コミュニティ機能 | 低 | 将来拡張 |
| ゲーミフィケーションの演出 | 低 | 将来拡張 |

## マスターDBの整備目標（Phase 0c での確定事項）

開発中はデータを絞ってよいが、リリース時点で以下を目標とする。

- 国内蒸留所: 50本
- 海外: 150本
- 合計: 200本

## レビュー結果（5観点）

- ✅ 各ステップとも「最小実装」で機能を限定し、進捗％表現になっていない
- ✅ 全機能が同じ1本のコアジャーニー（登録→記録→共有）に紐づいている
- ✅ 一人のユーザーが登録→記録→共有を順に体験できる順序になっている
- ✅ 後回し項目（特に高優先度のもの）が、ジャーニーの成立を妨げないことを確認済み
- ✅ PRFAQとコアジャーニー・ゴールラインの内容に矛盾はない
