# Phase 0: 要件の具体化・明確化

## プロンプト

```
これから作りたいシステムについて、私が思いつくままに話します。
まずは私の話をすべて聞いてください。途中で質問はしないでください。

私が「以上です」と伝えたら、以下を行ってください。
1. 話した内容を整理して、要件として分かりやすくまとめてください
2. まとめた内容に対して、曖昧な点・決まっていない点・矛盾している点を洗い出してください
3. 明確にすべき点を一度に一つずつ質問してください
4. 各質問には、あなたの推奨する回答も併せて提示してください

質問と回答を繰り返しながら、私たちが共通の認識に達するまで徹底的に要件を固めていきましょう。

では、話し始めてください。
```

---

## 使い方

このプロンプトを送信したら、作りたいシステムについて思いつくままに話してください。
箇条書きでも文章でも構いません。「以上です」と伝えるとAIが整理・質問を始めます。
